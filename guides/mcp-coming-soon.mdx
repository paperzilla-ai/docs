Paperzilla MCP support is planned but not yet available.

## What to use now

Use these interfaces today:

- [CLI](/guides/cli) for agent/script execution
- [RSS and Atom feeds](/guides/feeds) for polling-based integrations

## Planned direction

MCP support is intended to make Paperzilla easier to call from MCP-compatible LLM tools.

As this launches, this page will be updated with:

- availability status
- setup instructions
- supported operations
