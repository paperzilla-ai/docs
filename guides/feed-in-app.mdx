Each project has a live paper feed in the app.

## Open the feed

1. Go to **Projects**.
2. Open a project.
3. Click **View full feed**.

## Filter by priority

Use filter chips:

- **All**: includes `Must Read` and `Related`
- **Must Read**: highest-priority papers only

## Understand paper cards

Paper cards show:

- Title and authors
- Source (for example arXiv, bioRxiv, medRxiv, chinaXiv)
- Relevance class: **Must Read** or **Related**
- Relevance percentage
- Optional summary preview

## Search inside feed

Use search to filter by:

- Paper title
- Author names
- Summary text

## Open paper details

From a paper page, you can:

- Read metadata
- Open source or PDF links
- Generate or view AI summary
- Share (if project visibility allows)

## When the feed is empty

If a project is new or very narrow, you may see no papers yet.

Try:

1. Waiting for the next evaluation cycle.
2. Widening interest text slightly.
3. Removing overly strict exclude terms.
4. Enabling adjacent papers.
