Both.

For humans, Paperzilla provides app feed and email digests.

For agent or script workflows, Paperzilla provides CLI and RSS/Atom outputs that are easy to integrate today.

MCP integration is planned. See [MCP integration (coming soon)](/guides/mcp-coming-soon).

You can use both modes on the same project.
