Use one or more of these controls:

1. Lower **Max papers per email** in delivery settings.
2. Tighten project interest text.
3. Add targeted exclude keywords for noisy themes.
4. Set **Email frequency** to **Pause** to stop emails while keeping discovery running.
5. Pause the entire project to stop both discovery and emails.

You can still read from app, RSS/Atom, or CLI when you reduce email volume.
