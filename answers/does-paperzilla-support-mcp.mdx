Yes.

Paperzilla provides an MCP endpoint at:

`https://paperzilla.ai/api/mcp`

Use your per-user MCP API key in `Authorization: Bearer <key>`.

You can create or revoke this key from **MCP API key** in the top-right of the [dashboard](https://paperzilla.ai/digests).

Available tools today:

- `projects.list`
- `projects.get`
- `feed.get`
- `feed.atom_url`

There is also one optional prompt: `feed_title_filter`.

See setup and flow details in [MCP integration](/guides/mcp).
