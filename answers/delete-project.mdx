title: "How do I delete a project?"
description: "Delete a Paperzilla project from the project page using the delete action."
---

<AiAgents path="/answers/delete-project" />

1. Open the project.
2. Scroll to the delete section.
3. Click **Delete digest** (project delete action).
4. Confirm.

Deletion removes the project and related issues, papers, and settings.

This action is permanent.
