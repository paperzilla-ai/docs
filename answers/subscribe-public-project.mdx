---
title: "How do I subscribe to a public project?"
description: "Use Subscribe on a public project to preload setup and create your own project copy."
---

import { AiAgents } from '/snippets/ai-agents.mdx';

<AiAgents path="/answers/subscribe-public-project" />

1. Open a public project page.
2. Click **Subscribe**.
3. Review the preloaded setup in chat.
4. Adjust categories and keywords.
5. Save as your own project.

You are not modifying the original owner's project.
