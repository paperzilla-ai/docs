Yes.

Paperzilla MCP supports two per-paper tools:

- `paper.get` returns one paper's metadata and status fields
- `paper.markdown` returns that paper's markdown when it is ready

If markdown is not ready yet, `paper.markdown` returns a structured result that tells you whether markdown generation was queued or unavailable.

You can look up a paper by:

- paper UUID
- paper `short_id`
- feed item / `project_paper` UUID
- feed item / `project_paper` `short_id`
